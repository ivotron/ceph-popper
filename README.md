# Ceph

## Prerequisites
* Popper  
* Python 3  
* CloudLab credentials:  
  * CLOUDLAB_USER
  * CLOUDLAB_PASSWORD
  * CLOUDLAB_PROJECT
  * CLOUDLAB_PUBKEY_PATH
  * CLOUDLAB_CERT_PATH

## Installation 
* Clone the repo:  
``` 
git clone https://github.com/msouppe/Ceph.git
```
  
* Navigate to `Ceph/`  
  
* Run the program:  
```bash
poppper run
```
